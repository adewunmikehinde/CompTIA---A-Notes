# Hardware Basics 🖥️  

## Key Takeaways  
- Computers are made of input, output, processing, and storage components.  
- Examples:  
  - Input → Keyboard, Mouse  
  - Output → Monitor, Printer  
  - Processing → CPU  
  - Storage → HDD, SSD  

## Important Specs  
- CPU Speed (GHz)  
- RAM Size (GB)  
- Storage Capacity (TB/GB)
